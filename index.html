Internet Infrastructure | Te Aro

Wait what?

Internet infrastructure is a collective term for all the hardware and software systems that make up the components required to operate the internet. Physical cabling and routing equipment, as well as software services like DNS (Domain Name System), email, website hosting, storage systems, and database servers are all part of the this network.


The following is a collection of images taken around Te Aro, Wellington, depicting examples of components (or the location of components) that make up our internet infrastructure.

Hidden beneath certain inconspicuous manhole covers, an underground network of internet cabling exists.

The cabling that begins at your desktop computer connects to the modem, and eventually ends up connected to the cables that run underneath the manhole covers.
